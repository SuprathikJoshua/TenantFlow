
# Product Requirements Document (PRD)

## TenantFlow Frontend

---

## 1. Product Overview

TenantFlow Frontend is a lightweight web dashboard that consumes TenantFlow Backend APIs to demonstrate authentication, role-based UI behavior, and organization-scoped data access.

---

## 2. Target Users

- Owner
- Admin
- Member

---

## 3. Core Features

- Login
- Dashboard
- Task management

---

## 4. Tech Stack

- React
- TypeScript
- Fetch / Axios

---

## 5. Screens

### Login
- Authentication form
- Error handling

### Dashboard
- Organization name
- User role

### Tasks
- List tasks
- Create task
- Delete task (Owner/Admin)

---

## 6. API Rules

- Attach JWT to requests
- Handle 401/403 responses

---

## 7. Success Criteria

- Correct role-based UI
- Stable backend integration
